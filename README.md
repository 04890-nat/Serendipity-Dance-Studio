
# 動見舞蹈意識中心 (Moving Insight Dance Art Center) - 網站專案 (Vite + React)

## 1. 基本資訊 (Basic Information)

- **客戶名稱**：動見舞蹈藝術中心
- **網站目標**：讓更多的人認識動見舞蹈藝術中心，增加課程報名人數。
- **專案負責人**：吳煥然
- **技術架構**：React + Tailwind CSS + Vite
- **部署平台**：GitHub Pages
- **設計重點**：
  - **視覺呈現**：注重顏色搭配與高品質照片展示，營造優雅與專業感。
  - **使用者體驗**：直觀的課程篩選與清晰的報名流程。

---

## 2. 網站架構 (Website Architecture)

### 核心頁面與功能規劃

#### 1. 關於我們 (About Us)
- **品牌故事**：建立歷史感與穩定感，敘述創辦人的故事與中心沿革。
- **團隊介紹**：
  - 展示師資團隊，建立信任感。
  - 針對目標客群（如貴婦族群），強調教師風格與專業度，方便學員選擇。

#### 2. 課程資訊 (Class Information)
- **活動專區**：新增頁面展示中心定期舉辦的派對、舞會、品酒會或社交活動的照片與資訊。
- **課程列表與細節**：
  - 目標客群：主要為 18-40 歲。
  - 提供詳盡的課程細節，增強客戶信心並引導報名。
- **課程搜尋/篩選器 (關鍵功能)**：
  - **依舞風 (Type)**：國標舞、芭蕾舞、拉丁舞、街舞...
  - **依級別 (Level)**：基礎、初級、進階、大師班
  - **依教師 (Teacher)**：吳煥然老師、何柏蓁老師...
  - **依時間 (Time)**：晚間、週末、平日日間
- **課程卡片設計 (範例)**：
  - **主標題**：優雅華爾滋基礎班 (Waltz I)
  - **副標題**：專為入門打造，掌握舞池的浪漫轉身
  - **老師**：何柏蓁老師
  - **時間**：每週三 19:30-21:00 (1.5 小時)
  - **標籤**：試聽課、熱門課程、新開課程
  - **狀態**：尚有名額 / 即將滿班
  - **連結**：點擊查看詳情 / 立即報名

#### 3. 教師陣容 (Teachers)
- **展示內容**：
  - 教師個人照與形象照。
  - 個人資訊、教學風格介紹。
- **專業證明 (冠軍榮耀)**：
  - 特別展示吳煥然老師與舞伴在國際賽中的獲獎照片、影片、獎盃。
  - 何柏蓁老師的教學與比賽經歷，強化「吳老師」專業品牌形象。

#### 4. 學生成果 (Student Achievements)
- 展示高品質的學生成果展照片或影片。
- 突出學員的學習成效與優雅進步，作為強力的社群證明。

#### 5. 場地與設備 (Venue & Facilities)
- **位置**：新竹縣竹北市光明路 32 號 2 樓
- **設施特色**：
  - 專業加拿大楓木舞蹈地板
  - 舒適冷氣空調
  - 便利停車資訊
- **空間展示**：教室內部、休息區、更衣室照片。

#### 6. 舞風介紹 (Dance Styles)
- **國標舞**：
  - 摩登舞：華爾滋 (Waltz)、探戈 (Tango)、狐步 (Foxtrot)、快步 (Quickstep)
  - 英文華爾滋 (Viennese Waltz)
- **拉丁舞**：
  - 倫巴 (Rumba)、恰恰 (Cha Cha)、森巴 (Samba)、牛仔舞 (Jive)、鬥牛舞 (Paso Doble)
- **其他風格**：
  - 芭蕾舞 (Ballet)
  - 街舞 (Hip-Hop)
  - 現代舞 (Contemporary)
  - 爵士舞 (Jazz)

#### 7. 報名與收費 (Registration & Fees)
- **費用說明**：
  - 清晰列出單堂或課程價格 (例：第一堂課 NT$xxx)。
- **付款方式**：信用卡、轉帳、支付寶。

#### 8. 潛在客戶開發 (Lead Generation)
- 電子報訂閱功能。
- 免費資源專區（收集潛在客戶名單）。

#### 9. 聯繫我們 (Contact Us)
- 電話、Email、地址、營業時間。

#### 10. 常見問題 (FAQ)
- 回答關於課程內容、服裝準備、請假補課等常見疑問。
